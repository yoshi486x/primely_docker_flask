## Relative Docker Commands

**Build a docker image:**
```
$ docker build -t <image-name> <path/to/Dockerfile>
```

**Create a docker container:**
```
$ docker run -it <image-name-which-provided-during-image-build>
```

**Exit from a docker container**
* Mac:
`ctrl-D`
